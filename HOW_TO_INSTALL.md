# How to Install Vibe Security Enhanced

## Windows

Double-click `INSTALL.bat` and follow the prompts.

Or run it from the command line:

```cmd
INSTALL.bat
```

## Mac / Linux

```bash
chmod +x INSTALL.sh
./INSTALL.sh
```

## What the installer asks

1. Which AI assistant to install for (Kiro, Claude Code, Cursor, or all)
2. User-level (all projects) or project-level (one project)

That's it. Once installed, the skill activates automatically when you ask about security, request a code review, or work with auth/payments/databases.

To run a manual audit, just ask your assistant:

```
Run a security audit on this codebase
```
