# Changelog

All notable changes to the Superpowers Skills Framework will be documented in this file.

## [1.0.0] - 2026-03-24

### Added
- Initial release of Superpowers Skills Framework for AI assistants
- 6 core skills covering complete development lifecycle:
  - 00-using-superpowers.md - Framework overview
  - 01-brainstorming.md - Design before implementation
  - 02-test-driven-development.md - TDD workflow (RED-GREEN-REFACTOR)
  - 03-systematic-debugging.md - 4-phase root cause analysis
  - 04-verification-before-completion.md - Evidence-based verification
  - 05-writing-plans.md - Bite-sized task planning
  - 06-requesting-code-review.md - Pre-review checklists
- Installation scripts for Windows (INSTALL.bat) and Unix (INSTALL.sh)
- Support for multiple AI assistants (Kiro, Claude Code, Cursor)
- User-level and project-level installation options
- Comprehensive documentation (README.md, HOW_TO_INSTALL.md, QUICK_REFERENCE.md)
- MIT License

### Philosophy
- Test-Driven Development - Write tests first, always
- Systematic over ad-hoc - Process over guessing
- Complexity reduction - Simplicity as primary goal
- Evidence over claims - Verify before declaring success
